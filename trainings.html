Available trainings

Each project has different business requirements, just as each developer has different level of knowledge and experience. In order to make all attendees benefit the most, each training is preceded by a meeting with a tech representative (e.g. team leader), when we discuss the details of the program.

Training Description Duration
JavaScript Expert Course If you want to learn various aspects of the language, this training is the right one. It covers datatypes, asynchronous programming, communication via webservices, API mocking, automation, fundamentals behind modern frameworks, component architecture, and last but not least, best practices and common traps to avoid. 2-5 days
ECMAScript 6, 7, 8: new features ECMAScript 6 (2015) has brought lots of new features into the language and keeps on adding even more each year. Syntax extensions, new datatypes, promises, generators, iterators, coroutines and the Babel transpiler. Learn it all a TDD style. 2-3 days
Advanced TypeScript TypeScript introduces static typing to JavaScript, making it easier to reason about huge codebases and cheaper to maintain them. Learn the fundamentals, advanced techniques, best practices and possibilities of the platform, in order to improve your performance as a programmer. 2-3 days
Asynchronous JavaScript In JavaScript pretty much everything is asynchronous, and so we've got many ways to design the workflow. During this training, you'll learn strengths and weaknesses, usecases, best practices and gotchas about: callbacks, events, coroutines, async functions, CSP and observable streams. 2-5 days
RxJS: Observable Streams RxJS is the new way to design application flow in modern JavaScript. It relies on Functional Reactive Programming approach and requires developers to learn a completely new way of thinking about programing. 2-3 days
The Architectures of JavaScript This training puts much effort into explaining and understanding different paradigms and approaches of modern JavaScript. It includes many design discussions and exercises. 2-3 days
JavaScript for Backend developers If your team members are experienced in Java/.Net and want to raise their JavaScript skills, this training is for you. 2-3 days
Angular Learn one of the most popular frameworks of the frontend. Each aspect of the framework is explained during a lecture/discussion and followed by multiple exercises. Attendees build a full-featured web app throughout the training. 3-5 days

In order to discuss detailed agenda of your training, please contact me.
